# 💻 [scostadavid.github.io](scostadavid.github.io)

> My personal website written in Typescript with React and Gatsby.js


## Installation

```bash
$ yarn install
```

## Running the app
```bash
# development
$ yarn develop

# build static pages for deploy
$ yarn build
```
